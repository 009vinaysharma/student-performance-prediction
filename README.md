# 📌 Overview

AI Student Performance Prediction Dashboard is a Machine Learning web application that predicts a student's final grade and Pass/Fail status based on academic and personal attributes.

The project provides both:

- 👤 Single Student Prediction
- 📂 Bulk CSV Prediction

along with analytics, downloadable reports, and interactive visualizations.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/009vinaysharma/student-performance-prediction.git
```

Move into project

```bash
cd student-performance-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
